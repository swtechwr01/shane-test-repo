# API Endpoints Reference

## GET /api/v1/users
- Retrieves a list of users.

## POST /api/v1/users
- Creates a new user.

## GET /api/v1/users/{id}
- Retrieves a specific user by ID.

## PUT /api/v1/users/{id}
- Updates a specific user by ID.

## DELETE /api/v1/users/{id}
- Deletes a specific user by ID.
